Introducing JavaScript - changing images onclick

Using a HTML ID with JavaScript

A key part of working with JavaScript is undrstanding how JavaScript interacts with HTML. When you are trying to affect some part of your HTML you will use an ID to identify what you want to interact with. This is a critical concept. If you have been programming in HTML you may have wondered why to use an ID instead of just using classes for everything. It is important to know that you can only have one instance of an ID per page, so that will uniquely identify that portion of the page.

Changing images with JavaScript

  1. Edit your images and put them inside an images folder inside the folder for your HTML file
  2. Add your initial images and the CSS styles to the page
  3. Your large image - the one that you want to change when you click on a thumbnail, needs an ID - mine will be large
  4. Write your JavaScript (explained below)
  5. Call your JavaScript Function from each thumbnail when it is clicked

JavaScript code


function change01(){
document.getElementById("large").src="images/eli01.jpg";
}

function change02(){
document.getElementById("large").src="images/eliollie1.jpg";
}

function change03(){
document.getElementById("large").src="images/ender01.jpg";
}

function change04(){
document.getElementById("large").src="images/nessie01.jpg";
}

function change05(){
document.getElementById("large").src="images/nosey.jpg";
}

You declare the function with the word function, then provide a name. The .src= will be followed by the address of your image file.

Calling the JavaScript code

Inside of the image tag for each thumbnail, add the code: onclick="change01()"
the number will change for each thumbnail, so you will have change01(), change02(), etc.
